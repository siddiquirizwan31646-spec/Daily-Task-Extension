# Daily Task Extension 🚀

A simple and efficient browser extension designed to help you manage your daily tasks with ease. Stay organized, track your progress, and boost productivity—all in one place.

## ✨ Features

* ✅ Add, edit, and delete daily tasks
* 📅 Track tasks for each day
* 🔔 Simple and distraction-free interface
* 💾 Local storage support (no login required)
* ⚡ Fast and lightweight performance

## 🛠️ Tech Stack

* HTML
* CSS
* JavaScript

## 📦 Installation

1. Download or clone this repository:

   ```bash
   git clone https://github.com/your-username/daily-task-extension.git
   ```
2. Open your browser (Chrome recommended)
3. Go to Extensions → Enable Developer Mode
4. Click **Load unpacked**
5. Select the project folder
6. And Done Now Add Task

## 🚀 Usage

* Open the extension from the browser toolbar
* Add your daily tasks
* Mark tasks as completed
* Stay productive every day 💪
